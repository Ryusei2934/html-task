いかに接近して爆発するかがメインとなるこのファイターは、ストリートファイター６のゲーム性を象徴する存在と言える。

DETAILS

また、本作では代名詞のソニックブームとサマーソルトキックに「ジャスト入力」という概念が新たに加えられ、最後の前要素及び上入力要素と攻撃ボタンを同時に入力することで各種必殺技の性能が向上する。
